# 🔮 Crystal Ball Reader

Fortune teller application combining zodiac signs, Chinese zodiac, and tarot cards.

## Features
- 🌟 Zodiac readings (Western zodiac)
- 🐉 Chinese zodiac compatibility
- 🃏 Interactive tarot spread (3-card reading)
- 🌍 Multi-language support (Thai, English, Chinese)
- ✨ Beautiful crystal ball animation

## Languages Supported
- ไทย (Thai)
- English
- 中文 (Chinese)

## Tech Stack
- HTML5
- CSS3 (with animations)
- Vanilla JavaScript (no dependencies)

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Select your birth date and zodiac
4. Get your personalized fortune

## Deployment
Deployed on Vercel: [Live Demo](https://crystalball-reader.vercel.app)
